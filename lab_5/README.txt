Файл під назвою DS-lab5.1.py можна запускати по звичайному. Дана програма використовує matplotlib і працює в окремому вікні.
Файл під назвою DS-lab5.2.py використовує інструменти локального сервера бібліотеки для візуалізації даних bokeh. Для коректного запуску необхідно прописати $bokeh serve --show DS-lab5.2.py і програма відкриється в браузері.
